WHY NO .EXE?

Windows .exe files contain Windows-specific code that can only
be compiled on Windows. It's like trying to build an iPhone app
on Android - technically impossible without the right tools.

The good news: Everything is READY. Just run the build command
on any Windows 10/11 PC and you'll have your .exe in 5 minutes!
