üöÄ Installation Steps

Part 1: Desktop App Setup

1 Extract Files
  • Extract the copydock-app folder to a location of your choice
  • Recommended: C:\Program Files\CopyDock\
  • Or anywhere you prefer: C:\Users\YourName\CopyDock\
2 Run the App
  • Navigate to the copydock-app folder
  • Double-click START.bat
  • First launch will take 1-2 minutes (installing dependencies)
  • Subsequent launches will be much faster (~10 seconds)
‚úÖ If everything works, the CopyDock window will open automatically!

Part 2: Chrome Extension Setup

1 Load Extension in Chrome
  • Open Google Chrome
  • Type in address bar: chrome://extensions/
  • Turn ON "Developer mode" (toggle in top-right)
  • Click "Load unpacked"
  • Select the chrome-extension folder from this distribution
2 Test the Extension
  • Make sure desktop app is running
  • Go to any website (e.g., Wikipedia)
  • Select some text
  • A floating button "üìã Send to CopyDock" should appear
  • Click it to save the text
  • Check your desktop app - the content should appear!

üõ†Ô∏è Troubleshooting

Problem: "Python not found"

  • Reinstall Python with "Add to PATH" checked
  • Or manually add Python to PATH:
    • Search Windows for "Environment Variables"
    • Edit System Environment Variables
    • Add Python installation path to PATH variable

Problem: "Node not found"

  • Restart your computer after installing Node.js
  • Node.js should automatically add itself to PATH

Problem: Extension shows "Disconnected"

  • Make sure desktop app (START.bat) is running
  • Backend must be active on port 8001
  • Test by opening: http://localhost:8001/docs

Problem: Port 8001 already in use

  • Close any other apps using port 8001
  • Or edit copydock-app/backend/server.py and change the port number
