🚀 QUICK START (For Users):

STEP 1: Install Prerequisites
   → Python 3.8+: https://www.python.org/downloads/
   → Node.js 16+: https://nodejs.org/

STEP 2: Run Desktop App
   → Extract 'copydock-app' folder
   → Double-click START.bat
   → Wait for app window to open

STEP 3: Install Extension
   → Open Chrome: chrome://extensions/
   → Turn ON "Developer mode"
   → Click "Load unpacked"
   → Select 'chrome-extension' folder

STEP 4: Enjoy!
   → Browse any website
   → Select text
   → Click floating button
   → Text saved in desktop app!

🐛 TROUBLESHOOTING:

Common Issues:

1. "Python not found"
   → Reinstall Python with "Add to PATH" checked

2. "Node not found"
   → Restart computer after Node.js installation

3. Extension "Disconnected"
   → Ensure desktop app is running

4. Port 8001 in use
   → Close other apps or change port in backend/server.py

5. App doesn't open
   → Check backend.log in copydock-app folder
   → Ensure Python packages installed correctly
